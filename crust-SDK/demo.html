ℹ️ What This SDK Does

The Crust SDK silently collects behavioral signals from users without storing any personally identifiable information or cookies.

  • 40 behavioral signals extracted in real-time
  • Web Worker for off-main-thread collection
  • Privacy-first design — no cookies, no localStorage
  • Zero dependencies — pure TypeScript/JavaScript
